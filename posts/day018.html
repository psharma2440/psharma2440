Program 08 - Using for loop

Write a program to calculate the sum of series 2-4+6-8....N using for loop.

            
package main
import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	fmt.Println("Enter the value n upto your see the sum of series:")
	valueofN, _ := reader.ReadString('\n')
	valueofN = strings.TrimSpace(valueofN)
	intValuneofN, _ := strconv.Atoi(valueofN)
	sum := 0
	sign := 1
	firstTerm := 2
	for counter := 1; counter <= intValuneofN; counter++ {
		sum = sum + firstTerm
		firstTerm = counter * 2 * sign
		sign = sign * -1
	}
	fmt.Println("Sum of series:"sum)
}

Output:

            
                //Output
Enter the value n upto your see the sum of series:
500
Sum of series: 502

Explanation:

In this program, we calculate the sum of series 2-4+6-8.....+N
using (for loop). We take the value of 'N'from user as an input
upto which user want the sum of a series.
As an output we calculate the sum a series.
Accordingly to the user input value.
