Program - 4

Write a program to delete the element of a slice and calculate it`s length.

        
            package main

import "fmt"

func main() {
	x := []int{2, 3, 5, 7, 6}
	fmt.Println("Elements before delete of a slice 'x': ", x)
	fmt.Println("Lenght",len(x))
	// delete a slice
	x = append(x[0:2])
	fmt.Println("Elements after delete of a slice 'x':", x)
	fmt.Println("Lenght",len(x))
}

Output:

            
Elements before delete of a slice 'x':  [2 3 5 7 6]
Lenght 5
Elements after delete of a slice 'x': [2 3]
Lenght 2

Explanation:

In this program, we delete the elements of a slice 'x'.
And as an output we display the elements of a slice 'x' before and after delete the elements
and also compute the length of the slice 'x'.
