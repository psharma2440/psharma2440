Program 1 - Array

Write a program to count the odd and even number present in an array.

        
package main

import "fmt"

func main() {
	var numbers [10]int = [10]int{1, 11, 13, 5, 6, 7, 8, 9, 17, 24}
	evenCount := 0
	oddCount := 0
	for i := 0; i < 10; i++ {
		if numbers[i]%2 == 0 {
			evenCount += 1
		} else {
			oddCount += 1
		}
	}
	fmt.Println("The numbers we have:", numbers)
	fmt.Println("Total even count:", evenCount, "\nTotal odd count:", oddCount)
}

Output:

        
The numbers we have: [1 11 13 5 6 7 8 9 17 24]
Total even count: 3 
Total odd count: 7

Explanation:

Working with Array.
In this program, we are counting the number of odd and even number
present in an array.
