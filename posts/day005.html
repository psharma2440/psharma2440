goLanguage - Conditional Controlled Structure

Conditional Statement:

Conditional Controlled Structure may be define as it is a structure that is used to compare more than one quantites.
In Conditional Controlled Structure we have four types:
1) if Statement
2) if - else Statement
3) Nested if - else Statement
4) Switch case Statement

1) if Statement:

It is a Conditional Controlled Structure that is used to compare two quantites.

Syntax:

            
                      // Syntax:


                if Expression / Condition{
                 //Statements 
                }
                // if is keyword, Expression / condition may be variable, constant, arithmetic Expression or relational Expression 
                 Statement may be single Statement or compound Statement.
                 // compound Statement must be enclosed within braces({})

2) if - else Statement:

It is also a Conditional Controlled Structure that allows to execute any one of the two statement depending upon the values of Expression.

Syntax:

            
                     // Syntax:


                if expression / condition{
                    //statement1
                }else{
                    //statement2
                }
                // if the expression / condition is true statement1 is executed otherwise statement2 executed.

3) Nested if - else:

When one if-else structure is embedded into another if-else it is called nested if-else.

Syntax:

            
                      // Syntax:


                if  condition-1 { 
                    // code to be executed if condition-1 is true
                } else if condition-2 {
                    // code to be executed if condition-2 is true
                } else {
                    // code to be executed if both condition1 and condition2 are false
                }

4)Switch case Statement:

It is a conditional controlled structure i.e. used to select a statement among multiple statements depending upon value of expression.

Syntax:

            
                       // Syntax:


                Switch expression {
                    Case label 1:
                    Statement 
                    Case label 2:
                    Statement 
                    ……
                    ……….
                    Case label n:
                    Statement 
                    default: statements
                }
