reflect package

        
            package main

import (
	"fmt"
	"reflect"
)

func main() {
	fmt.Println("View the type of of value`s")
	// Return the type of it`s argument
	fmt.Println("1) 42:", reflect.TypeOf(42))
	fmt.Println("2) true:", reflect.TypeOf(true))
	fmt.Println("3) 1:", reflect.TypeOf(1))
	fmt.Println("4) 12.23:", reflect.TypeOf(12.23))
	fmt.Println("5) Hello Coder:", reflect.TypeOf("Hello Coder"))

}

Output:

        
            View the type of of value`s
            1) 42: int
            2) true: bool
            3) 1: int
            4) 12.23: float64
            5) Hello Coder: string

Explanation:

Working with reflect Package.In this program, as an output we can
view the type of value by passong it to reflect package`s TypeOf function.
