Program 10 - Using nested for loop

Nested looping:

When a loop is embedded into another loop it is said to be a nested looping.
In nested looping inner loop is executed completely again and again for each run of outer loop.

Write a program to show how one loop is embedded into another loop (Nested looping). 

            
package main

import (
	"fmt"
)

// nested loop
func main() {
	fmt.Println("Print a number:")
	for i := 0; i < 5; i++ {
		for j := 0; j < 10; j++ {
			fmt.Println("i:", i, "j:", j)
		}

	}
}
