"Write a program to find largest / greater number between two numbers".

            
                
package main

import "fmt"

func main() {
    var num1 int = 20
	var num2 int = 11
	fmt.Println(num1 > num2)
	if num1 > num2 {
		fmt.Println("num1 is greater than num2")	
	}
	if num1 < num2 {
		fmt.Println("num2 is greater than num1")
	}
}

Output of the Program:

            
                // Output of the Program: "Write a program to find largest / greater number between two numbers".

                num1 is greater than num2

Explaination of the program:

In this we have two inputs i.e. num1 and num2.


Variable Syntax:

            
                // Variable Syntax:
                         keyword        variable(identifier)            datatype   =  value
                 Example:   Var           num1                           int       =   12

After assign the value to input we check which number is largest. So, in this we use If Conditional Controlled Structure:- It is a Conditional Controlled Structure that is used to compare two quantites. In our program we have two quantites i.e. num1 and num2. If num1 is largest / greater than num2 then first if statement is execute otherwise if second statement is execute.
If statement Syntax: 

             
                 // Syntax:
                 if Condition {
                     // statement
                 }

// if is a keyword. Expression / condition may be variable, constant, arithmetic Expression or relational Expression.
 // statement may be single or compound statement.

"Write a program to find largest / greater number between two numbers and input the numbers from user."

    
        package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	fmt.Println("Enter a number1:")
	number1, _ := reader.ReadString('\n')
	number1 = strings.TrimSpace(number1)
	intNumber1, _ := strconv.Atoi(number1)
	fmt.Println("Enter a number2:")
	number2, _ := reader.ReadString('\n')
	number2 = strings.TrimSpace(number2)
	intNumber2, _ := strconv.Atoi(number2)
	fmt.Println("Numbers entered by user:", intNumber1, intNumber2)
	if intNumber1 > intNumber2 {
		fmt.Println("number1 is largest than number2")
	}
	if intNumber1 < intNumber2 {
		fmt.Println("number2 is largest than number1")
	}
}
