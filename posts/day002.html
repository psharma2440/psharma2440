At Day001, I learn few terminal commands, that i explain to you in details; with their Full-Form, Uses, Expalination and Exapmle.

1. Cd:-

Full-Form:

Change directory.

Use:

Use to change the directory.

For Example:

$ cd .. 

                
                    // Command cd ..

        
                    xyz@workbook:~/projects$ cd .. 
                    xyz@workbook:~$ pwd
                    /home/xyz

Expalination of cd.. command:

In terminal if i type command cd .. which is shown you above, then i reach at parent directory i.e. xyz. To see what is your parent directory, that command will explain you later.

2. pwd:-

Full-Form:

Print working directory / Present working directory.

Use:

Use to Print working directory / Present working directory.

For Example:

$ pwd 

                    
                        // Command pwd
                        
                        
                        xyz@workbook:~$ pwd
                        /home/xyz

Expalination of pwd command:

In terminal, when i type command pwd command, it show me the present working directory which is shown in above i.e., my present working directory is "xyz".

3.ls:-

Full-Form:

list

Use:

It is used to list files and directories.

For Example:

$ ls 

                
                    // Command ls 

        
                    xyz@workbook:~/projects$ ls
                    projects my_book photo

Expalination of ls command:

In terminal, when i type command ls, then it list my all the files and directory that are shown you above.
