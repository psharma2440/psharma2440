Two-Dimensional(2D) Array

Introduction:

What is 2D Array?

Two-Dimensional(2D) Array:-
The two-dimensional array can be defined as an array of arrays.
And it is organized as matrices which can be represented as the collection of rows and columns i.e [m x n] where m = rows;
n = columns

Declaration of two dimensional Array:

Keyword variable name [no. of elements in rows] [no. of elements in columns] data type value
var numbers [5] [5] int [5][5]int{ {12,23,34,56,70}, {2,3,4,1,5}, {4,5,6,3,5}, {1,2,3,4,5}, {21,23,45,26,28}, }

Explanation:

In this var is the Keyword.
numbers is the name of the variable.
[5][5] is the no. of elements in rows and columns.
And int is the data type. Indexing of rows and columns starts from 0.
[5][5]int{
{12,23,34,56,70}, // 0 row
{2,3,4,1,5}, // 1 row
{4,5,6,3,5}, // 2 row
{1,2,3,4,5}, // 3 row
{21,23,45,26,28}, // 4 row
}

Accessing Two-Dimensional Array Elements

An element in two dimensional array is accessed by using the subscripts, i.e., row index and column index of the array. For example from above example- x := numbers[1][2]
In this it print the value = 3
Because it take the value from 1 row & column 2. So the value at 1 row 2 column is 3.
