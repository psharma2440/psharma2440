goLanguage Program - Using for loop


For loop



Looping:-Repeated execution of a set of statement for a define no. of time is called looping.


goLanguage has only one loop i.e. for loop.


The basic for loop has three components separated by semi-colons.
1. The init statement: Executed before the first iteration.
2. The condition expression: Evaluated before every iteration.
3. The post statement: Executed at the end of every iteration.
The init statement will often be a short variable declaration, and the variables declared they are visible only in the scope of for statement.
The loop will stop iterating once the Boolean condition evaluates the false.

Write a program that prompts the user to input a positive integer and then print the multipilcation table of that number.

        
package main
import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	var result int
	fmt.Println("Enter a positive integer:")
	positiveInteger, _ := reader.ReadString('\n')
	positiveInteger = strings.TrimSpace(positiveInteger)
	intPositiveInteger, _ := strconv.Atoi(positiveInteger)
	fmt.Println("user enter a positive integer:", intPositiveInteger)
	fmt.Println("Multiplication of table of",intPositiveInteger,":")
	for counter := 1; counter <= 10; counter++ {
		result = intPositiveInteger * counter
		fmt.Println(result)
	}

}
