Swaping of the numbers

Write a program to swap two numbers.

    
        package main

import "fmt"

func main() {
	var a int = 20
	var b int = 10
	fmt.Println("Value of a before swaping:", a)
	fmt.Println("Value of b before swaping:", b)
	var c int
	c = a
	fmt.Println("value of c", c)
	a = b
	fmt.Println("Value of a after swaping", a)
	b = c
	fmt.Println("value of b after swaping", b)
}

Output:

    
Value of a before swaping: 20
Value of b before swaping: 10
value of c 20
Value of a after swaping 10
value of b after swaping 20

Explanation:

In this program, we swap the two numbers.
As an output we display both the cases
1) Values before swaping.
2) Values after swaping.
