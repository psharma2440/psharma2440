Program - 2

Write a program to compute the sum of the two array.

        
            package main

            import "fmt"
            
            func main() {
                var arr1 [2][2]int = [2][2]int{
                    {2, 3},
                    {4, 5},
                }
                var arr2 [2][2]int = [2][2]int{
                    {2, 3},
                    {4, 5},
                }
                fmt.Println("Element in first array =", arr1)
                fmt.Println("Element in second array =", arr2)
                var sum [2][2]int
                for i := 0; i < 2; i++ {
                    for j := 0; j < 2; j++ {
                        sum[i][j] = arr1[i][j] + arr2[i][j]
                    }
                }
                fmt.Println("Sum of first and second array element`s =", sum)
            }

Output:


        
            Element in first array = [[2 3] [4 5]]
            Element in second array = [[2 3] [4 5]]
            Sum of first and second array element`s = [[4 6] [8 10]]

Explanation:

In this program, we compute the sum of two array:
arr1 and arr2;
After that store the result of sum in sum array.
And then as an output we display the element`s of arr1, arr2 and sum array.
