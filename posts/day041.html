Formatting Verbs

Write a program to display different types of verb with their Output type using Printf function.

        
            package main

import "fmt"

func main() {
	fmt.Println("Formatting verbs\n")
	// first field have min. width of 12 character and no min. width of second field character
	fmt.Printf("%12s | %s\n", "Verbs", "Output")
	fmt.Println("---------------------------------") // printing a heading divider
	// first field have min. width of 5 character and min. width of second field character is 20
	fmt.Printf("%5s | %20s\n", "%f", "Floating point")
	fmt.Printf("%5s | %20s\n", "%d", "Decimal integer")
	fmt.Printf("%5s | %20s\n", "%s", "string")
	fmt.Printf("%5s | %20s\n", "%t", "Boolean(True/False)")
	fmt.Printf("%5s | %20s\n", "%%", "A literal percent sign")
}

Output:

            
Formatting verbs

       Verbs | Output
---------------------------------
   %f |       Floating point
   %d |      Decimal integer
   %s |               string
   %t |  Boolean(True/False)
   %% | A literal percent sign

Explanation:

In this program, we display the different types of formatting verbs like,
%f, %s, %t, etc with their output value using Printf function.
