Program - 3

Write a program to add elements of slice and calculate lenght.

        
            package main

import "fmt"

func main() {
	var name [][]string = [][]string{
		{"A", "N", "S", "H"},
		{"P", "R", "F", "P"},
	}
	fmt.Println("Before Append:\n", name)
	fmt.Println("Length", len(name))
	name = append(name, []string{"d", "e", "f"})
	fmt.Println("After Append:\n", name)
	fmt.Println("Length", len(name))
}

Output:

            
 Before Append:
 [[A N S H] [P R F P]]
Length: 2
After Append:
 [[A N S H] [P R F P] [d e f]]
Length: 3

Explanation:

In this program, we use the append function to add the elements in the slice (name).
As an output we display the slice before and after use of append function
and also compute it`s lenght using len function.
