String package- replacer

        
            package main

import (
	"fmt"
	"strings"
)

func main() {
	var a string = `Hell$! 
	#g$lang #g$pher #c$ding #learng$ #day035`
	fmt.Println("Before use of string package replacer type:\n", a)
	replacer := strings.NewReplacer("$", "o")
	fixed := replacer.Replace(a)
	fmt.Println("After use of string package replacer type:\n", fixed)
}

Output:

        
            Before use of string package replacer type:
            Hell$!
                   #g$lang #g$pher #c$ding #learng$ #day035
           After use of string package replacer type:
            Hello!
                   #golang #gopher #coding #learngo #day035

Explanation:

Working with String Package- replacer.In this program, as an output we can
replace the $ dollar symbol wiht letter o.
