goLanguage Program - Using if-else statement


Write a program to find the largest number between two number.

            
                
package main

import "fmt"

func main() {
	var age int = 17
	fmt.Println(age >= 18)
	if age >= 18 {
		fmt.Println("you are eligible for the vote")
	}

	var num1 int = 20
	var num2 int = 11
	
	if num1 > num2 {
		fmt.Println("num1 is greater than num2")	
	}else {
		fmt.Println("num2 is greater than num1")
	}
}

Output:

            
        
                // Output:
                num1 is greater than num2

Explaination of the code:

if-else is also a Conditional Controlled Structure that allows to execute any one of the two statement depending upon the values of Expression. In this we have two statement in this if num1 is greater than num2 then first statement execute otherwise else statement is executed.

"Write a program to find largest / greater number between two numbers".

            
                package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	fmt.Println("Enter a number1:")
	number1, _ := reader.ReadString('\n')
	number1 = strings.TrimSpace(number1)
	intNumber1, _ := strconv.Atoi(number1)
	fmt.Println("Enter a number2:")
	number2, _ := reader.ReadString('\n')
	number2 = strings.TrimSpace(number2)
	intNumber2, _ := strconv.Atoi(number2)
	fmt.Println("Numbers entered by user:", intNumber1, intNumber2)
	if intNumber1 > intNumber2 {
		fmt.Println("number1 is largest than number2")
	}else{
		fmt.Println("number2 is largest than number1")
	}
}

Output:

            
                // Output:
                Enter a number1:
2
Enter a number2:
5
Numbers entered by user: 2 5
number2 is largest than number1

Explaination of the code:

if-else is also a Conditional Controlled Structure that allows to execute any one of the two statement depending upon the values of Expression. In this we have two statement in this if user enter two numbers. if number1 is largest than number2 then first statement executed i.e. number1 is largest than number2 otherwise else statement is executed i.e.number2 is largest than number1.
