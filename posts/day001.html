Introduction:

1)Problem solving process:


Enitre process is divided into 4 parts:
Sr.No. Name Description
1 Formulation of problem We specify output required and then decide number of minimum Input required to get the specify Output.
2 Deriving method of solution In this step, we define the method of solution of the given problem and there are two technique; Algorithm, Flow chart.
1) Algorithm: Stepwise representation of method of solution to any problem is called algorithm.
2) Flow chart: Diagrammatical or pictorial representation of method of solution to any problem is called flow chart.
3 Coding It refers to encode the method of solution into computer language usually high-level language.
4 Debugging and Execution Debugging refers to removal of errors in the code. In this step, the code is feed into the computer, compiler removers the errors if any and then program is executed.

Algorithm: Stepwise representation of method of solution to any problem is called algorithm.

For Example:

                    
                        //Algorithm: To find area of reactangle.
                        1)Formulation of the problem:
                            a)Output of the problem- To find the Area of the reactangle(area).
                            b)Minimum no. of input require- minimum two input required i.e., length, breadth.
                        2)Input length, breadth.
                        3)Calculate area.
                        4)Area of the reactangle (area) = length * breadth
                        5)Output area.

Flow chart: Diagrammatical or pictorial representation of method of solution to any problem is called flow chart. It consists of some specific diagrams that were given by American National Standard Institute (ANSI).

Flow chart
Sr.No. SymbolName Description
1 oval Every flow chart must have a start in beginning and stop statement at the end.
2 Parallelogram It is used of any kind of input and output.
3 Rectangle This is used for process/calculation.
4 Rhombus This is used to take decision in terms of True or False.
5 Circle It is used to Joint the same two part of the same flow chart.
6 Lines These are the straight lines with arrow heads. These are used to connect various diagram in the flow chart. Thus, it shows the direction of flow control.
