Basic Introduction to Array.

Derived data type:-

These are non-primitive data type which are composed of one or more one fundamental data type.
These are derived from fundamental data type.
For example: struct, array, slices.

Array

An array is a collection of finite number of homogenous items.
The homogenous means that all the data items of an array must be a same data type.
The array elements are stored in consecutively memory location one after another.
By default, the value of each array element is either 0, nil(string), False/ True(Boolean).

Array declared syntax:

Keyword variable name [no. of elements] data type value
var numbers [5] int [5]int{12,23,34,56,70}

Example: 

                    
                    var a[5]int = [5]int{2,3,5,6,7}

Accessing the array elements-

Accessing of the array elements- The element in an array identified by their position in the array.
In go positions (index value) of first element is 0
and second element is 1
and third element is 2 and so on.
Each element is referred by array name followed by it`s index value
and enclosed within square brackets.
