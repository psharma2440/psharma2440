Program 02 - Using for loop

Write a program to print the sum of Positive series 1+2+3+.....n 

            
package main
import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	fmt.Println("Enter the value of 'n' upto which you want the sum positive series:")
	valueofn, _ := reader.ReadString('\n')
	valueofn = strings.TrimSpace(valueofn)
	intvalueofN, _ := strconv.Atoi(valueofn)
	sum := 0
	for count := 1; count <= intvalueofN; count++ {
		sum = sum + count
	}
	fmt.Println("Sum of,intvalueofN,Positive series:", sum)
}

Output:

            
                //Output:
Enter the value of 'n' upto which you want the sum of positive series:
10
Sum of 10 Positive series: 55

Explanation:

In this program, we calculate the sum of first natural number
"or" Positive series 1+2+3+.....n known as natural number
using (for loop). We take the value of 'n'from user as an input
upto which user want the sum positive series.
As an output we calculate the sum of positive series.
Accordingly to the user input value.
