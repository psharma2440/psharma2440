Slice-

A slice is a segment of an array.
Like arrays slices are indexable and have a length.
Unlike arrays this length is allowed to change.

Example:-

             
                var x []int

Explanation:

The only difference between this and an array is the missing length between the brackets.
In this case x has been created with a length of 0.

Slice Functions

Go includes two built-in functions to assist with slices:
1) append 2) copy
